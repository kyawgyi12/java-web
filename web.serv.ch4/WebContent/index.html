JDC Servlet Day 4

In this tutorial we will learn about Database connectivity by JDBC. And then we will learn how to upload a file by servlet. Finally we will learn Asynchronous Support by servlet.

Student Registration

In this sample, we use a form page for student input, a servlet that get user inputs and add a row to table. After that the servlet redirect to another servlet that show students from database.

Student File Upload

In this sample, a csv file is uploaded from html form. After uploading this file, csv is parsing into java object and data will be insert into database. After inserting data, forward to the list page.

Asynchronous Update

In this sample, we are using asynchronous request function of servlet. It start from the list page and if you click edit button, the update form will be open by asynchronous process. After that if you edit and press save button. The form will be submit to servlet and update the database.
